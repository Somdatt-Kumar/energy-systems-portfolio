# Solar PV System Modelling

## Problem
Estimate real-world energy output of a grid-connected solar PV system.

## Why it Matters
Accurate PV modelling is essential for grid planning, forecasting, and system optimization.

## Methodology
- Physics-based PV performance equations
- Temperature and irradiance loss modelling
- Validation using real-world solar data

## Data
Publicly available solar radiation and weather datasets.

## Expected Outcomes
- PV output estimation
- Loss sensitivity analysis
- Engineering insights for performance improvement
